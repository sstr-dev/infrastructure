# Namespace: `llm`

This namespace contains local AI, search, and document-assistant style services.

## Components

| Application | Description | Links |
|-------------|-------------|-------|
| [open-webui](./open-webui/) | Web UI for local or remote LLM backends | [Website](https://openwebui.com/) [GitHub](https://github.com/open-webui/open-webui) |
| [paperless-ai](./paperless-ai/) | AI-assisted workflows around Paperless-managed documents | [GitHub](https://github.com/clusterzx/paperless-ai) |
| [paperless-gpt](./paperless-gpt/) | LLM-assisted document interaction for Paperless content | [GitHub](https://github.com/icereed/paperless-gpt) |
| [searxng](./searxng/) | Metasearch engine often used alongside AI assistants | [Website](https://docs.searxng.org/) [GitHub](https://github.com/searxng/searxng) |
